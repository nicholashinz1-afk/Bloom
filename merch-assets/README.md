## How to convert SVG to PNG for Printful

Printful needs PNG (transparent background) or JPG files. These SVGs need to be converted:

### Option 1: Browser export
1. Open the SVG in Chrome
2. Right-click > Inspect > Console
3. The SVG will render at the correct pixel dimensions
4. Use a screenshot tool or browser extension to export at full resolution

### Option 2: Figma (recommended)
1. Import the SVG into Figma
2. The font (Fraunces) is a Google Font, so Figma will load it automatically
3. Export as PNG at 1x (the SVG is already at the right pixel dimensions)
4. Make sure background is transparent

### Option 3: Inkscape (free, offline)
1. Open SVG in Inkscape
2. Install Fraunces font from Google Fonts first
3. File > Export PNG Image
4. Set DPI to 300
5. Export

## Printful upload specs
- Format: PNG (transparent background) for DTG printing
- Resolution: 300 DPI recommended, 150 DPI minimum
- Color profile: sRGB IEC61966-2.1
- For hats: Upload the embroidery file, Printful handles digitization
